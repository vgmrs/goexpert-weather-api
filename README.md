# goexpert-weather-api
Exercise Weather API for postgraduate Go Expert

## Description

This application implements a Weather API using Clean Architecture structure in Golang.

## How to run

### Using Golang:
```bash
make run
```

### Using Docker:
```bash
make container
```

### Using Docker Composer:
```bash
make up
```
And for down containers:
```bash
make down
```

### To run tests:
```bash
make test
```
